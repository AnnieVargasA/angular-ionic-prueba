# Prueba Ionic Angular

Aplicación de prueba desarrollada con Ionic Framework y Angular.

## Descripción

Este proyecto es una aplicación base creada con Ionic y Angular para fines de prueba y desarrollo.

## Requisitos

- Node.js (versión 14 o superior)
- npm o yarn
- Ionic CLI (`npm install -g @ionic/cli`)

## Instalación

```bash
# Clonar el repositorio
git clone <url-del-repositorio>

# Instalar dependencias
npm install
```

## Desarrollo

```bash
# Ejecutar en modo desarrollo
ionic serve
```

La aplicación se abrirá automáticamente en `http://localhost:8100`

## Build

```bash
# Build para producción
ionic build --prod
```

## Estructura del Proyecto

```
src/
├── app/              # Componentes principales de la aplicación
│   ├── home/        # Página de inicio
│   └── ...
├── assets/          # Recursos estáticos
├── environments/    # Configuración de entornos
└── theme/          # Estilos globales
```
